# Tome

Share shell spells.

## Development

Note: To run, you will need to provide a config file (default: `~/.tome.<EXTENSION>`).
For supported EXTENSIONs see the ![viper doc](https://github.com/spf13/viper#what-is-viper).

To run the code showing the help text:
```
go run .
```

To run `last` command:
```
go run . last
```
